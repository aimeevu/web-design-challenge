Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this, we used OpenWeatherMap API to pull data on over 500 cities

Once the dataset was ready, we used Matplotlib to plot different aspects of the weather and latitide. Some factors that we included in our analysis were temperature, cloudiness, wind speed, and humidity.

This site provides the source data as well as the visualizations that were created as part of this analysis.
